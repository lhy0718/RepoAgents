# Merge Policy

- Default merge mode is `human_approval`.
- Supported publication modes are `comment_only`, `draft_pr`, and `human_approval`.
- RepoRepublic does not auto-merge changes in the MVP.
- Reviewer approval means the patch is ready for a maintainer to inspect, not that it should merge automatically.
- Docs and tests work can be marked as lower-risk candidates and can stay comment-only or open a draft PR, but the final merge action remains manual.
